_Хеш-атака_. Найдите $2^N$ строк, каждая длиной $2^N$, которые имеют одинаковое значение _hashCode()_ для следующей реализации:

```
public int hashCode(){
    int hash = 0;
    for(int i=0;i < length();i++){
        hash = (hash*31)+charAt(i);
        return hash;
    }
}
```
